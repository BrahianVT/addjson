# **App Name**: Promos

## Core Features:

- Produce Browsing: Browse available fruits and vegetables.
- Price Viewing: View current prices for each produce item.
- Price Filtering: Filter items by price range.
- Tag-Based Search/Filter: Search and filter produce by descriptive tags like 'organic' or 'local'.
- Upload Product: A form is available for users to upload products and an admin should approve it.

## Style Guidelines:

- Primary color: Fresh green (#90EE90) to evoke nature and freshness.
- Background color: Light pastel green (#F0FFF0), a very lightly desaturated tint of the primary hue.
- Accent color: Soft yellow (#FAFAD2), an analogous color for highlighting prices and tags.
- Font: 'PT Sans', a humanist sans-serif, suitable for headlines or body text.
- Use clean, modern icons to represent different produce categories.
- Clean and simple layout with clear separation of items and filters.
- Subtle transitions when filtering or sorting produce items.
